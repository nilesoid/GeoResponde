# Security Policy

## Supported Versions

Currently, GeoResponde is in alpha testing. Only the latest release on the `master` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a vulnerability, please do NOT report it by creating a public GitHub issue.

Instead, please email the security team at **security@georesponde.org** (or the maintainer's email) to disclose it responsibly.

We will acknowledge your report within 48 hours, and provide a timeline for triage and resolution.
