# Organizations Knowledge

This folder contains detailed profiles, contact strategies, trust evaluations, and historical context for the organizations (NGOs, Government, Private, etc.) that provide or curate geospatial and humanitarian data. 

Use this folder to document qualitative research on an organization before adding them as a structured entry in the catalog.
